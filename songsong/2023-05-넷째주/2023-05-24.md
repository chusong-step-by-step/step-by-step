# Element.classList

    - Element.classList.add(string) : 클래스 이름 추가
    - Element.classList.remove(string) : 클래스 이름 제거
    - Element.classList.contains(string) : 클래스 값 존재 유무 확인
    - Element.classList.contains(old, new) : 클래스 값 변경(old->new)
